<h1>It's forms!!</h1>
  • refers to the part of a document used to give input/submit information
  • has an action attribute which details where the data should be sent
  • The method attribute specifies which http method should be used
  • The forms take input using the "input" elements demarkated as input tags
  • There are more than 20 different input types each of which alter the type of input dramatically
  • to change the type, you use the "type" attribute
  • input tags have no closing tags

    it has to be inside the form element.
